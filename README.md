# 새싹 개발자를 위한 ICT 인턴십 서울기업 가이드

개발자로서 **인생 첫 인턴**을 앞둔 후배에게, 선배가 "여기 어때?" 하고 소개해 주는 가이드입니다.
과기정통부 ICT 이노베이션스퀘어 인턴십 **서울 소재 64개 기업**을 정리했습니다.

👉 **[가이드 바로 보기](https://windowispark.github.io/ict-intern-seoul-guide/)**

## 무엇을 보나
스펙·연봉이 아니라 첫 인턴에게 진짜 중요한 것으로 판단했습니다.
- **배울 사수가 있는가** · **회사가 안정적인가** · **잡무가 아닌 실무를 맡는가**
- 기업별 **한줄평 / 얻는 경험 / 이런 후배에게 좋아요 / 예상 경쟁(추측)**

등급: 🟢 추천 · 🟡 확인 필요 · ⚪ 신중

## ⚠️ 면책 (꼭 읽어주세요)
- 한줄평·등급·예상 경쟁은 **AI 에이전트가 공개정보(뉴스·투자이력·잡플래닛·블라인드 등)를 조사해 만든 참고용 추정**입니다.
- 사실과 다르거나 시점이 지난 정보가 있을 수 있으며, **특정 기업을 폄훼할 의도가 없습니다.**
- **지원 전 반드시 각 기업의 공식 채용정보를 직접 확인하세요.**
- 정정 요청은 이슈로 남겨주시면 반영하겠습니다.
